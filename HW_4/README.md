Задача №3

Необходимо вывести все числа кратные 4 между числами 40 и 60 включительно.
Реализовать 2 варианта: 
1) использовать конструкцию if для определения кратности (цикл с шагом 1, i = i + 1);
2) без использования конструкции if (шаг цикла на ваше усмотрение).

_____________________________________________________________

P.S. Решение задачи №3 пункт 2 от преподавателя

public class qwerty {
    public static void main(String[] args) {
        for (int i = 40; i <= 60; i = i + 4){
            System.out.println(i);
        }
    }
}

Решение действительно выводит числа кратные 4 в промежутке от 40 до 60 без использования конструкции if, НО это работает только при шаге равном 4. Если шаг цикла будет отличаться от 4, то нам просто будет выдаваться список чисел в указанном диапазоне с выставленным шагом цикла, что по моему мнению не соответствует условию "(шаг цикла на ваше усмотрение)". Решение задачи предоставленное мной проверяет кратность чисел в указанном диапазоне с любым шагом цикла, формально без использования конструкции if напрямую, но получилось добиться этого с помощью тернарного оператора ? : который в свою очередь является сокращенной формой конструкции if else. Без использования тернарного оператора в System.out.print в результате мы получим только тип boolean (true, false) вместо числа, программа прощитывает числа в диапазоне с заданным шагом и присваивает true либо false в зависимости соответствия условию кратности и выводит нам только список с типом boolean, а не int. Можно в print вывести для наглядности следующее: System.out.println("a = " + a + ", делится на 4? " + (a % 4 == 0));
